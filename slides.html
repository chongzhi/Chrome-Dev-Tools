chrome的几个版本

  • chrome(Stable, dev, beta)

    http://www.chromium.org/getting-involved/dev-channel

    • stable，稳定版，推荐追求稳定的普通用户使用，更新最慢。
    • beta，测试版，有一定新功能，但是可能会存在不稳定情况，适合喜欢尝鲜的朋友使用，更新速度一般。
    • dev，开发版，更新最快，新功能最多，但是可能非常不稳定，适合开发人员使用。
  • chrome canary (金丝雀版)

    http://tools.google.com/dlpage/chromesxs

  • chromium (chrome就是基于这个开源项目发展而来)

    http://www.chromium.org/getting-involved/download-chromium

一个google帐号可同步所有信息(收藏夹，插件等)

Developer Tools概述

Web App

  1. Elements     DOM节点查看，样式动态修改等
  2. Resoures     列表出网站所有资源，包括了html5支持的Database, localstorage等
  3. Network     查看各资源的网络加载情况，详细请求信息等
  4. Scripts     脚本面板可让您调试自己的 JavaScript 代码
  5. Timeline     详细分析你的网页或应用程序时间用在了何处，包括资源加载，js解析，页面渲染重绘等
  6. Profiles     帮你分析网页或应用的执行时间和内存使用情况
  7. Audits     类似yslow的建议分析器
  8. Console     强大的控制台，无须赘言

官方参考文档：http://code.google.com/intl/zh-CN/chrome/devtools/docs/overview.html#access
英文原版：http://code.google.com/intl/en/chrome/devtools/

快捷键

在打开chrome developer tools情况下，按 "?" 键调出

比较好用常用的几个快捷键

  • 各面板下ESC调出console面板，方便引用当前元素
  • console面板：ctrl+L，清除控制台信息
  • resourse, script面板：ctrl+g，跳转行
  • script面板：ctrl+o列出所有js文件，ctrl+shift+o列出当前js文件所有function,支持字符筛选
  • ctrl+shift+F: 在所有js文件中搜索，支持正则，忽略大小写
  • 断点相关：F8暂停/继续，F10跳到下个function，F11单步进入，Shift+F11单步退出

Settings

  • disable cache : 禁用缓存，开发时方便调试
  • Show paint rectangles : 是否显示渲染范围框
  • Override User Agent : 覆盖用户代理
  • Text editor : 控制格式化js时缩进规格
  • ......

Elements

google 演示网页

  • DOM面板 任意节点修改删除 , F2
  • Computed style 查看最终样式计算结果
  • Styles 实时修改style [ ]，可另存
  • Metrics 查看盒模型，可实时修改
  • Properties 以对象形式查看元素，包括继承的类
  • DOM Breakpoints 设置的dom节点断点【演】
  • Event Listeners 查看各dom节点上的事件侦听(事件代理) 【例】

Resource

  • Frames 列出所有资源(html, css, js, images)可进行全局搜索
  • Databases html5本地数据库，对象本地持久化

    http://www.webkit.org/demos/sticky-notes/index.html【演】

  • IndexedDB
  • Local Storage html5的本地存储 【例】
  • Session Storage 会话存储
  • Cookie 查看当前页面的cookie

Network & Audits

  • HTTPRequest 查看详细的http请求信息，包括request method, headers, response headers
  • Timeline 每条请求详细的发起，响应，接收时间，以及window loaded, content load时间线
  • JSON preview 【例】
  • Audits chrome上的yslow
    • Network Utilization
    • Web Page Performance

Scripts

  • Script format 格式化压缩代码，让代码更可读
  • Pause on exception 自动在异常处断点
  • Live edit script 直接编辑js文件，ctrl+s保存，只对用户事件有效【例】
  • Watch Expressions 断点处任意查看变量或表达式【例】
  • Call Stack 查看函数的调用堆栈【例】
  • Scope Variables 查看局部变量，包括Local, Global, Closure 【例】
  • Breakpoints DOM Breakpoints, xhr Breakpoints, EventListener Breakpoints【例】
    • 带表达式的Breakpoints
  • Workers html5 多线程
